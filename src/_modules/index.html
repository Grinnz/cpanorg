How to install modules

There are several methods you can use, see the Installing Perl Modules page.

How to find modules

  • http://search.cpan.org/ - search
  • Task::Kensho - some recommended modules
  • Browse:
    • All modules (a long list)
    • Authors
    • Name
    • Category (not maintained)
    • recentness

Questions about the modules? Please contact the module author directly. Corrections? Additions? Suggestions? Please contact cpan@perl.org. Other questions? See the CPAN FAQ.
